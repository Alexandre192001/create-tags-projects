<h2>⚜️Descriçao⚜️</h2>
<p>Criador de tags para linguagens.</p>
<hr>
<h2>⚜️Funcionalidades:⚜️</h2>
<ul>
<li>Enter - cria um campo para tag quando ela estiver com um texto dentro dela;</li>
<li>X - Remove a tag.</li>
</ul>
